# Okta Notes

Okta is a **cloud-based IAM platform**.

---

## What Okta Does
- Single Sign-On (SSO)
- Multi-Factor Authentication (MFA)
- User lifecycle management
- Application integration

---

## Okta in IAM Flow

HR System → Okta → Applications

Example:
- HR adds employee
- Okta creates user
- Access is given automatically

---

## Okta Features
- Central identity store
- Secure login
- Integration with AD & LDAP
- Cloud and on-prem apps support

---

## Benefits of Okta
- Improved security
- Reduced manual work
- Better user experience
- Centralized access control
